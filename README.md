# 🚀 Voice Calculator  
A modern Voice-Controlled Scientific Calculator Web Application built using **Angular** and **Node.js**.  
The application allows users to perform real-time mathematical calculations using both manual input and voice commands.  

---

### 🔗 Live Demo:  
[https://voice-calculator-theta.vercel.app/](https://voice-calculator-theta.vercel.app/)  

---

### ✨ Features:  
* Voice input support using Web Speech API  
* Scientific calculator functions (sin, cos, tan, log, ln, sqrt, π, e, etc.)  
* RAD / DEG angle mode toggle  
* Dark / Light mode toggle  
* Real-time API-based calculation  
* Keyboard support (Press Enter to calculate)  
* Ripple button animation  
* Smooth result fade-in animation  
* Calculation history tracking  
* Fully deployed production setup
